# Temi Engine

## 1: About
Screen Based Platformer engine with a variety of its own formats. Implimented in python via pygame. Created for the Express Purpose of Making Al' Sugani Temi.

## 2: Why Python?
Python was picked for a variety of reasons:
- Python is the language for those who do not know what they are doing. I do not know what I am doing.
- Many libraries that can be used to solve problems (ex. Pygame, Numpy).
- I only need to be able to do everything in 15ms, plenty of time even for the less performant python.
- Python works on any machine that also has Python on it. I don't need to compile for every system I could dream to target.
